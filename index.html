Procedural Terrain Generator

An old repository made during my undergraduate for PIC10C: Advanced Programming. Uses C++ and the Qt Framework. My team-member forked our private repository as a public one (and also wrote a nice description in the readme).

The project compiles into a GUI application that generates a terrain using a very simplified climate model and an extremely simplified tectonic (meaning literally just procedural noise). Idea was heavily inspired by games such as Civilization and Dwarf Fortress.

Repository was made while both of us didn't understand either git or GitHub well: it was never the focus of the class, we just tried to find some way to collaborate. The project itself is not a great example of how to structure a C++, nor a great example of procedural terrain generation. I think it at least showcases us trying our bests to get something working with a framework that we were both unfamiliar with.
